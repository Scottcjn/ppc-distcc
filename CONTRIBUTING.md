# Contributing to PPC DistCC

Distributed compilation system for PowerPC Macs (Tiger/Leopard/Sorbet).

## Development Setup

```bash
git clone https://github.com/MingYu5/ppc-distcc.git
cd ppc-distcc
```

## Supported Systems

- Mac OS X Tiger/Leopard PowerPC
- Python 2.5 to 3.10 compatible

## Contributing

1. Fork the repo
2. Create a feature branch
3. Test on real hardware
4. Open PR against `Scottcjn/ppc-distcc:main`

Questions? Open an issue!
